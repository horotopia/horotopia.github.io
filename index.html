Veille Technologique

Qu'est-ce que la veille technologique ?


La veille technologique consiste à surveiller les évolutions techniques, les innovations dans un secteur d’activité donnée.
La veille technologique comprend notamment la surveillance, la collecte, le partage et la diffusion d’information permettant d’anticiper ou de s’informer sur des changements en matière de recherche, développement, brevet, lancement de nouveaux produits, matériaux, processus, concepts, innovation de fabrication, etc….

Les principaux outils de ma veille


J'ai effectué ma veille technologique grâce aux sites d'actualités tel que daily.dev, frandroid, etc.
Les YouTubeurs comme Underscore_, Graven, Le Frère Codeur, Captain Dev, Hugo Dev Club, etc.
Pour ce qui est des informations matériel (processeurs, cartes graphiques, cartes mères, ect.), je me rends sur certains sites : Hitek, Hardware, lesnumeriques.

Sujet de ma veille


Pour ma veille technologique, j'ai choisi la programmation mobile multi-plateforme.
Lorsque l’on parle mobile, on parle de plusieurs supports, les principaux étants Android et Ios.
Chacun peut être coder dans différents langages :
Kotlin et Java pour Android, Objective-C et Swift pour Ios.
Mais pour entreprise, cela pose soucis car s’ils veulent apparaitre sur tous les supports, il faudra programmer non pas un code unique mais un code pour chaque support.
Pour cela, il est possible d’utiliser l’hybrid ou le multi-plateforme.
L’hybrid étant du web, il n’utilise pas les performances du smartphone.
Le multi-plateforme est donc la meilleure solution.
